# GitHub Setup Assignment

## Description
This project was created as part of a course assignment to practice using GitHub. The goal of the assignment is to learn how to create a repository, edit files directly on GitHub, use Markdown, and commit changes properly. This repository demonstrates basic GitHub workflow skills rather than a full software application.

## Tools Used
- GitHub  
- Markdown  
- Web Browser (Chrome)

## Files Used
- None

## How to Run Program
There is no program to run for this project. The repository is used for learning GitHub setup and documentation practices.

## Additional Information
This repository was created for academic purposes. All work was completed following course guidelines, and GitHub is being used as a learning tool to understand version control and project documentation.
